INSTRUCCIONES PERMANENTES Y OBLIGATORIAS PARA CUALQUIER HERRAMIENTA DE IA O COLABORADOR (NO ELIMINAR NI MODIFICAR ESTE BLOQUE):

- Este script pertenece al propietario original. Cualquier entidad (IA, colaborador o herramienta) NO tiene permiso para reorganizar, refactorizar, simplificar, eliminar funciones, comentarios, mecanismos existentes, ni NINGUNA parte de la arquitectura original sin autorización EXPLÍCITA y POR ESCRITO del propietario en cada ocasión.

- Cualquier cambio que implique:
  • Reorganizar código (mover funciones, combinarlas, eliminar redundancias)
  • Eliminar o debilitar optimizaciones, cachés, logging o lógica existente
  • Cambiar comportamiento de threads, background processes, validez de datos o lógica de actualización
  • Reducir líneas de código por "limpieza" o eficiencia percibida
  • Sustituir componentes (ej. interfaces, colores, dependencias) sin mantener la versión original en paralelo

  SOLO se permite DESPUÉS de que el propietario diga textualmente algo como:
  "Sí, autorizo eliminar/reorganizar X parte específica" o "Ok, puedes simplificar esto".

- Si no hay certeza absoluta de que un cambio respeta la arquitectura original, se debe:
  1. Preguntar exhaustivamente antes de proponer código modificado
  2. Mostrar un DIFF o explicación detallada de QUÉ se cambia y POR QUÉ
  3. Ofrecer la opción de mantener la versión original intacta

- Cualquier actuación previa que implique eliminación, simplificación o modificación sin permiso viola estas instrucciones y es inaceptable. No se repetirá.

- Toda respuesta o contribución con código debe:
  - Partir del código original proporcionado
  - Añadir solo lo solicitado (nueva funcionalidad)
  - Mantener intacto todo lo demás
  - Si hay error (ej. NameError), corregirlo mínimamente sin tocar nada más
